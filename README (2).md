## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/mehulsarrraf1234/ai-ml-projects.git
cd ai-ml-projects
```

### 2. Install Dependencies
```bash
# Image Classifier
cd 01_Image_Classifier
pip install -r requirements.txt
python main.py

# Sentiment Analysis
cd ../02_Sentiment_Analysis
pip install -r requirements.txt
python train.py

# Predictive Analytics
cd ../03_Predictive_Analytics
pip install -r requirements.txt
streamlit run app.py

# Recommendation System
cd ../04_Recommendation_System
pip install -r requirements.txt
python recommendation_engine.py
```
