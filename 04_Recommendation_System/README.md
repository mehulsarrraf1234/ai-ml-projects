# 🛒 E-commerce Analytics & Recommendation System

A complete e-commerce analytics solution with recommendation engine, market basket analysis, and KPI dashboards.

## 📌 Project Overview

This project implements:
- **Collaborative Filtering** recommendation engine
- **Market Basket Analysis** using association rules
- **E-commerce Analytics** dashboard
- **User behavior analysis**
- **Sales trend analysis**

## 🔧 Features

- Collaborative filtering (user-based & item-based)
- Association rule mining (Apriori algorithm)
- Sales analytics and reporting
- Customer segmentation
- Product recommendation engine
- Interactive visualizations

## 📊 Components

1. **Recommendation Engine**: Suggests products based on user behavior
2. **Market Basket Analysis**: Identifies product associations
3. **Analytics Dashboard**: KPI tracking and visualization

## 🛠️ Installation

```bash
pip install pandas numpy matplotlib seaborn scikit-learn mlxtend plotly
```

Or use the provided requirements.txt:
```bash
pip install -r requirements.txt
```

## 🚀 Usage

### Run the Recommendation System:
```bash
python recommendation_engine.py
```

### Run Market Basket Analysis:
```bash
python market_basket_analysis.py
```

### Run Analytics Dashboard:
```bash
python analytics_dashboard.py
```
